GETCVSDATA

#Le projet consiste à prendre des données dans un fichier au format cvs.

Les fonctionnalités du projets sont les suivantes:

1)Récupérer les données;
2)Les affichers dans un tablea html;
3)Calculer le total net;
4)Calculer le total des dépenses;
5)Calculer le total des révenus;
6)Afficher les révenus en vert;
7)Afficher les dépenses en rouges;
8)Afficher tout les montant précéder du signe ($);
9)Afficher la date au format (Jour / Mois / Année).
